Detailed Description
--------------------

For undirected graphs stored as directed adjacency lists, this adaptor provides efficient access to reverse (back) edges. Given edge (u,v), get_back_edge returns a reference to the corresponding edge (v,u).

This is particularly useful for algorithms that need to update both directions of an undirected edge, such as max-flow or min-cut algorithms.
