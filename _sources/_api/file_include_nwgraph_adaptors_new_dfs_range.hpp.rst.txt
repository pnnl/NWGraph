Detailed Description
--------------------

Provides DFS range adaptors with additional features:

- dfs_range: Vertex iteration in DFS order

- dfs_edge_range: Edge iteration with tree/back edge classification



The edge range yields tuples (is_tree_edge, source, target, ...) allowing algorithms to distinguish between tree edges and back edges.

Cormen, Leiserson, Rivest, and Stein, "Introduction to Algorithms", 4th Edition (2022), Chapter 20.3: Depth-First Search
