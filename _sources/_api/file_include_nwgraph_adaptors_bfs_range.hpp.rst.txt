Detailed Description
--------------------

This file provides range-based interfaces for BFS traversal over graphs:

- topdown_bfs_range: Traditional top-down BFS from a seed vertex

- bottomup_bfs_range: Bottom-up BFS traversal



These adaptors allow BFS traversal using C++ range-based for loops: for(autov:topdown_bfs_range(graph,start_vertex)){
//ProcessvertexvinBFSorder
}


Cormen, Leiserson, Rivest, and Stein, "Introduction to Algorithms", 4th Edition (2022), Chapter 20.2: Breadth-First Search
