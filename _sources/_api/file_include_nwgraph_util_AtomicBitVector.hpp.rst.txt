Detailed Description
--------------------

Provides AtomicBitVector, a concurrent bitmap for use in parallel graph algorithms. Supports atomic get/set operations with configurable memory ordering. Commonly used for visited arrays in BFS and similar traversal algorithms.

Example usage: #include<:ref:`file_include_nwgraph_util_AtomicBitVector.hpp`>

//Createavisitedarrayfor1000vertices
:ref:`exhale_class_classnw_1_1graph_1_1AtomicBitVector`visited(1000);

//Thread-safecheck-and-setpatternforBFS
size_tvertex=42;
if(!visited.atomic_set(vertex)){
//Firstthreadtomarkthisvertex-processit
process_vertex(vertex);
}

//Checkifavertexwasvisited(relaxedorderingforread-only)
if(visited.get(vertex)){
//Alreadyvisited
}

//Iterateoverallsetbits(non-zerovertices)
for(autov:visited){
std::cout<<"Visited:"<<v<<std::endl;
}

//Clearallbitsforreuse
visited.clear();
