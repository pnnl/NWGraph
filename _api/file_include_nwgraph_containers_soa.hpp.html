Detailed Description

Provides struct_of_arrays, a container that stores tuples as separate contiguous arrays (SoA layout) rather than an array of tuples (AoS). This layout improves cache performance when accessing single fields across many elements, which is common in graph algorithms.

Example usage: #include<File soa.hpp>

//Createastruct-of-arrayswithvertexIDandweight :ref:`exhale_struct_structnw_1_1graph_1_1struct__of__arrays`edges; edges.push_back(1,0.5); edges.push_back(2,1.5); edges.push_back(3,2.0);

//Accessastuples for(auto&&[target,weight]:edges){ std::cout<<”->”<<target<<”(weight:”<<weight<<”)”<<std::endl; }

//Accessindividualarrays(cache-efficientforsingle-fieldtraversal) auto&targets=std::get<0>(edges); auto&weights=std::get<1>(edges);
