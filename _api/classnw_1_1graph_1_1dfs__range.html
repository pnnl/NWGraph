template<typename Graph, typename Stack = std::stack<vertex_id_t<Graph>>>
class dfs_range

Range adaptor for depth-first search vertex traversal.

Provides a range-based interface for visiting vertices in DFS order. The traversal continues to unvisited components if the graph is disconnected.

Note

The range can only be traversed once (single-pass).

Template Parameters:

  • Graph – The graph type to traverse.

  • Stack – The stack type for DFS (default: std::stack).
