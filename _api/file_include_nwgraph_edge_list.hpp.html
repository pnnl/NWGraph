Detailed Description

Provides the index_edge_list class template and its convenience aliases for storing edges before converting to adjacency format. Edge lists are the primary input format for building CSR graphs.

Example usage: #include<File edge_list.hpp> #include<File adjacency.hpp>

//Createadirectededgelistwith5vertices :ref:`exhale_class_classnw_1_1graph_1_1index__edge__list`edges(5); edges.push_back(0,1); edges.push_back(0,2); edges.push_back(1,2); edges.push_back(2,3); edges.push_back(3,4); edges.close();//Finalizetheedgelist
