template<typename Graph, typename Stack = std::stack<vertex_id_t<Graph>>>
class dfs_edge_range

Range adaptor for depth-first search edge traversal.

Provides a range-based interface for visiting tree edges in DFS order. Each iteration yields a tuple representing the edge (source, target, …).

Note

The range can only be traversed once (single-pass).

Template Parameters:

  • Graph – The graph type to traverse.

  • Stack – The stack type for DFS (default: std::stack).
