template<class Range, class Op, class Reduce, class T>
auto nw::graph::parallel_reduce(Range &&range, Op &&op, Reduce &&reduce, T init)

NWGraph’s parallel_for wrapper that supports reductions.

This performs the designated reduction over the range. It will perform the reduction in parallel if range.is_divisible(), otherwise it will perform a synchronous sequential reduction.

Template Parameters:

  • Range – The type of the range.

  • Op – The type of the operator.

  • Reduce – The type of the reduction.

  • T – The reduced type.

Parameters:

  • range – The range to process.

  • op – The operator to evaluate.

  • reduce – The reduction.

  • init – The initial value for the reduction.

Returns:

The result of reduce(op(i), ...) for all i in range.
