Detailed Description

Provides timer classes for measuring execution time:

  • timer: Basic start/stop/elapsed timer with configurable duration type

  • life_timer: RAII timer that prints elapsed time on destruction

  • Type aliases: seconds_timer, ms_timer, us_timer

Example usage: #include<File timer.hpp>

//Basictimerusage :ref:`exhale_class_classnw_1_1util_1_1timer`timer; timer.start(); //…worktomeasure… timer.stop(); std::cout<<”Elapsed:”<<timer.elapsed()<<”ms”<<std::endl;

//Laptimer(stopandgetelapsedinonecall) timer.start(); //…firstphase… doublephase1=timer.lap(); //…secondphase… doublephase2=timer.lap();

//RAIItimer-printselapsedtimewhenitgoesoutofscope { nw::util::life_timerlt(“BFStraversal”); //…BFScode… }//Prints:”#[BFStraversal]:123.45ms”

//Usingostreamoperator :ref:`exhale_class_classnw_1_1util_1_1timer`named_timer(“Dijkstra”); named_timer.start(); //…work… named_timer.stop(); std::cout<<named_timer<<std::endl;//”(Dijkstra)45.2ms”
