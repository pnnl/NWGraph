template<class Iterator>
class splittable_range_adaptor : public std::ranges::view_base

Splittable_range_adaptor can split a range into sub_views of a range.

This adaptor wraps a range and provides the interface required by parallel backends (TBB’s Range concept, or HPX’s iteration patterns) to split work across multiple threads.
